## üìñ Usage (Unchanged)

All APIs remain identical to v0.0.5:

```javascript
import Ombor from 'ombor'

const db = new Ombor('myDatabase')

// Add data
await db.collection('users').add({
  name: 'Otabek',
  age: 25
})

// Get data
const users = await db.collection('users')
  .orderBy('age', 'desc')
  .limit(10)
  .get()

// Update data
await db.collection('users')
  .doc({ name: 'Otabek' })
  .update({ age: 26 })

// Delete data
await db.collection('users')
  .doc({ name: 'Otabek' })
  .delete()
```
